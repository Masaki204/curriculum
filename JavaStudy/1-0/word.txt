■ IT

IT 
ITとは「Information Technology」の頭文字をとった単語。パソコンやスマートフォンなどの情報技術の総称を意味する。ITという単語はパソコン、インターネットを表すだけでなくセキュリティや情報処理なども含み広義的な意味で使われる。

ICT 
ICTは「Information and Communication Technology」の頭文字をとった単語で、ITをほぼ同じ意味を持つが、ICTは情報技術だけでなく、その情報や技術を共有するための「コミュニケーション」の意味がより強調された用語である。

IoT
　「Internet of Things」の略称で身の回りのあらゆるモノがインターネットに繋がる仕組みを表している。従来のパソコンやスマートフォンだけでなく、テレビやエアコン、冷蔵庫などの身の回りのあらゆるものがインターネットに繋がることで、モノが相互通信し、遠隔から認識や計測、制御をすることが可能になった。

■ OS

OS　
OSとは「Operating System」の略でコンピューターにおいて、ハードウェアを機能毎に抽象化したインターフェースを利用者またはアプリケーションソフトウェアに提供するコンピュータープログラムである。

Windows　
WindowsはMicrosoft社が開発したOSである。パソコンで使用されているOSの中で世界で最もユーザーが多いOSである。

macOS (Mac OS X)
Mac OsはApple社が作ったOSである。

UNIX　
UNIXは現存する中で最も古いOSの一つで様々なOSの土台となったOSである。特徴は安全性が高く、また余計な機能が少ないので軽いことが特徴である。

Linux　
LinuxはWindowsやMacが有料に対して、無料で使えるOSで誰でも自由に改良できるという特徴がある。


■ 言語・環境

マークアップ／フロントエンド
マークアップとは、コンピュータによって処理される人工言語の種類の一つで、データ中に特定の記法を用いて何らかの情報を埋め込むためのもの。テキストデータ中に特定の記号で囲まれたタグと呼ばれる表記を用いて構造や見栄えなどを記述するものがよく知られるが、バイナリデータ中に埋め込むものなど、様々な種類がある。
フロントエンドは、WebサービスやWebアプリケーションで直接ユーザーの目に触れる部分のこと。

HTML
HTMLとは、マークアップ言語の一つ。ホームページを作る時に使う。「Heper Text Markup Language」

CSS
CSSとは、Webページの要素の配置や見栄えなどを記述するための言語。HTML文書に追加した見た目をコントロールすることができ、文書の外部から読み込んで適用することもHTMLデータ中に埋め込んで記述することができる。

JavaScript
JavaScriptとは、ホームページに動きをつけるためのプログラミング言語。

サーバーサイド
サーバーの内部で動き、フロントエンドのプログラムからの指示を待ち受けて、処理を行うもの。
.NET（C#, VB）
C#はマイクロソフト社が開発してプログラミング言語であり、C＋＋やJavaといった世界的に使われている言語と同じ「オブジェクト指向」を採用している。オブジェクト指向とは、独立性、再利用性、拡張性を実現するためのプログラミング言語である。
VBは「Visual Basic」の略でマイクロソフト社によって開発され、主にWindows用のアプリケーションを開発するためのプログラミング言語である。BASICをベースとしており、Windowsとの親和性を高く保っており、全般的にコーディングをしなくても比較的に安易にプログラミングできるのが大きな特徴である。

Java
Javaはプログラミング言語の一つで、世界的に見ても使用されている言語の一つである。大きな特徴としては、汎用的でありOSに依存しない点である。Webアプリケーションのサーバーサイドプログラムを組むことができるため、Webとの相性も良く、どんなものにも搭載されているマルチな言語である。

Java SE
Java SEとは「Java Standard Edtion」の略で、Javaの基本機能をまとめたものである。

Java EE（Jakarta EE）
Java EEとは「Java Enterprise Edtion」の略で、Java SEを元にしてサーバーサイドの機能を追加したものである。

Perl
Perl(パール)とは、「ラリー・ウォール」によって開発されたプログラミング言語である。実用性と多様性を重視しており、C言語やsed、awk、シェルスクリプトなど他のプログラミング言語の優れた機能を取り入れている。ウェブ・アプリケーション、システム管理、テキスト処理などのプログラミングを書くのに広く用いられている。

PHP
PHPの正式名称は「PHP：Hypertext Preproceccor」で、主にオープンサイドで動的なコンテンツの作成に向いているプログラミング言語である。動的なコンテンツとは、同じURLであっても閲覧者の属性や時間帯によって表示される内容が変わってくるコンテンツのこと。代表的なものはブログで多くの場合、PHPが使用されている。

Ruby
Rubyとは日本で開発されたオブジェクト指向スプリクト言語で、Webアプリ開発やスマホアプリ開発などオールラウンドに使える言語である。特徴は汎用性が高く、記述がシンプルであるなどが挙げられる。


■ サーバーソフト

Apache
Apacheとは、WebサイトをPCやスマートフォンに提供するWebサーバーソフトウェアでシェア率は世界でも最も多いと言われている。多くの人に使われている理由は、安定していて機能が豊富であり、動作環境が多彩であるからである。

nginx
Nginx(エンジンエックス)はApacheと比較して、高速かつ高負荷に強いのが特徴。大量のリクエストを処理するのに向いている。

IIS
IISはマイクロソフト社が開発したWebサーバーソフトウェアで「Microsoft Internet Infomation Services」の略である。
